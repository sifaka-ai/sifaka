Supervised Learning:** In this method, we provide the model with a set of input-output pairs and the model learns the mapping function from this labeled data to program itself. The most common tasks using supervised learning are classification (where the outputs are discrete labels, for example, spam or not-spam) and regression (where the outputs are continuous, for example, house prices). The most common tasks in unsupervised learning include clustering (grouping similar instances together), dimensionality reduction, and associative rules. Example: Suppose we have data about different customers in a supermarket but no specific labels. Unsupervised learning can help in grouping customers into clusters (like family shoppers, impulsive buyers, etc.) based on their shopping behaviors. **When to use:** Unsupervised learning is beneficial when we want to find hidden structures or patterns in the data, but we don't have a specific prediction task in mind. **3. Reinforcement Learning:** Reinforcement learning is about learning from feedback. The model, called an agent, interacts with an environment and learns to make decisions by receiving rewards or punishments. The goal of the agent is to learn a policy, which is a strategy to select an action that maximizes the total reward over time. Example: One practical example is training a machine to play a video game. The agent (machine) learns from each game it plays. If it wins a game, it gets a reward, and if it loses, a penalty. Over time, the machine will 'learn' which actions increase its chances of winning. **When to use:** This approach is often used when we need to make a series of decisions and the final goal is far from the immediate next step (for instance, playing a chess game).
